Identifying User Requirements and Worker Engagement Through Iterative Prototypes
Master's Thesis

Akademiska Sjukhuset is in the early stages of identifying requirements for a new software system for the pharmacists. By creating prototypes and interviewing the pharmacists, we can find user requirements needed for the system. Will we also be able to assess user motivation with the prototypes and identify how to improve user motivation?
